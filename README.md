# airbnb-imitations 爱彼迎WEB移动端仿写
**说明：该项目仅作为项目演练，不作为任何商业用途**

## 技术栈
Vue + Vuex + Vue-router + Koa + MongoDB + Axios （参考Airbnb编写的原生UI）
